# 🍦 Ice Cream Paradise

A fun and visually appealing ice cream website built using **HTML, CSS, and JavaScript**.
This project started as a basic static website and was later enhanced with interactive features to improve user experience.

---

## 🚀 Features

* 🎨 Beautiful and responsive UI
* 🔍 **Search functionality** to find your favorite flavors
* 🌙 **Dark mode toggle** for better viewing experience
* 🛒 **Add to cart (basic interaction)**
* 🍦 **Loading animation** for smoother feel
* ✨ Smooth scrolling and hover animations

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript 

---

## 💡 What I Learned

* How to structure a frontend project properly
* Separating HTML, CSS, and JavaScript
* Adding interactivity using JavaScript
* Improving UI/UX with small features like animations and dark mode
* Deploying a website using Netlify
